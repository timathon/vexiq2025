5.2 强迫症患者的自我修养

8月10日

现在的机器人上不仅有电线，还有黑色的气管，乱得像一碗炸酱面。

叶恒瑞受不了了，拿了一大把扎带，开始理线。

理线规则：

  • 电线走左边，气管走右边，井水不犯河水。
  • 翻转关节那里的气管要绕成螺旋状（像电话线那样），不然一转头就把管子拔断了。

理完之后，看着整整齐齐的走线，感觉战斗力都提升了 50%！

5.3 编程：气动怎么写代码？

8月12日

叶恒瑞打开电脑，突然懵了。以前控制爪子是 `motor.spin`，现在气动怎么搞？

查了半天资料才发现，气动电磁阀在代码里叫 Digital Out (数字输出)。

// 这就是控制空气的魔法代码！
digital_out Solenoid = digital_out(PORT10);

// 按下 R1 键，发射空气！
if (Controller.ButtonR1.pressing()) {
Solenoid.set(true); // 爪子闭合！
}
else if (Controller.ButtonR2.pressing()) {
Solenoid.set(false); // 爪子张开！
}

写完下载进去，按下 R1，“咔哒”一声，爪子合上了。全场欢呼！
