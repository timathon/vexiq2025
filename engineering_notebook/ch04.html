4.1 机器人要干什么活？

根据之前的规则分析，我们把机器人的工作列了个清单。这就像是给机器人找工作的“招聘启事”。

招聘：全能机器人

岗位职责：
1. 跑腿：在场地上跑得快，还得停得稳。
2. 搬运工：能抓起圆圆的 Pin，也能拖动重重的 Beam。
3. 建筑师：能把 Pin 一个个叠起来（Stack）。
4. 杂技演员：能把倒着的 Pin 翻过来（Flip）。

叶恒嘉： “这活儿也太多了吧！而且还要缩在那么小的盒子里出发，太难了！”

4.1 数学题：6 减几等于几？

我们遇到了第一个大难题：电机不够用！

规则 <R11> 说最多只能用 6 个智能电机。

我们的算计：
底盘（左右轮子）： -2 个
抬升手臂（举重）： -1 个（起码吧？）
抓手（开合）： -1 个
翻转手腕： -1 个
剩余： 1 个

宋书月： “我还想给底盘装 4 个电机呢，那样推力大，不怕撞。”

印涵邸： “不行不行，那就没电机做手臂了！我们是去得分的，不是去撞碰碰车的。”

最终决定： 省着点花！底盘只用 2 个电机。把剩下的资源留给那些复杂的动作。

4.2 怎么抓那个滑溜溜的Pin？(1)

Pin 是圆柱形的，表面很滑，这是最头疼的问题。

方案 A：铲子 (Fork)

原理： 像叉车一样，直接插到 Pin 底部的缝隙里。

缺点： 只能铲起来，但是如果不小心一晃，Pin 就会掉下来。而且没法把 Pin 翻转过来。

大家评价： 太简陋了，Pass！

记录人：印涵邸
69

4.2 怎么抓那个滑溜溜的Pin？(2)

方案 B：滚筒吸入 (Roller)

原理： 上下两个滚轮转动，像吃面条一样把 Pin 吸进去。

叶恒瑞说： “这个效率最高！我看网上的大神都用这个。”

叶恒嘉的担忧： “但是这个结构好大啊！而且吸进来之后怎么叠在一起呢？Pin 在肚子里乱滚，怎么对准？”

[图片：草图，画了一个复杂的滚筒结构，旁边打了个大大的问号“怎么堆叠？”]

我们讨论了半天，觉得这就是“看着美好，做起来难”。对于一代车来说太复杂了。

记录人：印涵邸

4.2 怎么抓那个滑溜溜的Pin？(3)

方案 C：主动夹爪 (Claw) - 最终选择

原理： 像螃蟹钳子一样，或者像人的两根手指，把 Pin 夹住。

优点：
1. 抓得紧。
2. 可以保持 Pin 是直立的，方便堆叠。
3. 可以在爪子上装旋转轴，实现翻转。

[图片：草图，画了一个两指夹爪夹住 Pin 的样子，标注了“稳”]

虽然一次只能抓一个，比较慢，但是我们要的是准确度！只有抓稳了才能堆高高。

记录人：印涵邸

4.2 怎么举高高？(1)

我们要把 Pin 举起来，挂到高塔上，或者叠在别的 Pin 上面。

方案 A：四连杆 (4-Bar Linkage)

原理： 用四根梁组成一个平行四边形。

优点：
1. 举起来的过程中，爪子始终保持水平，不会把 Pin 倒出来。
2. 结构简单，这也是 VEX 这里的经典结构。

记录人：印涵邸

4.2 怎么举高高？(2)

方案 B：双反四连杆 (DR4B)

原理： 两个四连杆叠罗汉！

优点： 能举得超级高！而且是直上直下的。

缺点： 太重了！太复杂了！我们怕做出来车头太重，一刹车就翻跟头。

力气够不够？

Beam 很重（170克），再加上爪子，我们担心一个电机举不起来。

力矩 = 力 x 距离
手臂长：30 厘米
重量：约 300 克 (3 牛顿)
需要的力矩： 0.9 牛·米

可是VEX IQ电机只有大约 0.4 牛·米的力量。完蛋了，举不动！

解决方案： 1. 减速齿轮： 用 12齿 带动 60齿，力气变大 5 倍！
2. 橡皮筋： 在手臂后面挂橡皮筋帮忙拉。

记录人：印涵邸
78

神器的橡皮筋

我们在手臂根部挂了几根橡皮筋。这真是太神奇了！

效果： 就算不通电，手臂也能自己悬在半空中，不下坠。这就叫“重力补偿”。电机只需要用一点点力气就能指挥手臂上下动了。

4.2 最难的挑战：Beam

Beam 那么长，那么重，如果用前面的爪子抓，车头会很重，容易翻车。而且爪子很难对准 Beam 的重心。

方案 A：后置拖钩 (Passive Hook)

想法： 在车屁股后面装个钩子。倒车，钩住 Beam，然后拖着走。

优点： 不用电机！结构超级简单。

缺点： 只能在地上拖，没法举起来。

记录人：印涵邸
80

4.2 最难的挑战：Beam

方案 B：主动翻转爪

想法： 让前面的爪子能翻转到底下，把 Beam 铲起来。

缺点： 太复杂了，而且我们没有多余的电机做这个翻转动作了。

叶恒嘉： “我们一代车先别想那么多了，先把 Beam 拖到得分区就能拿 10 分，这已经很好了！”

决定： 先做简单的后置拖钩。

记录人：印涵邸
81

简单的拖钩

虽然简单，但也要设计好。

1. 高度： 钩子不能太低，不然会蹭到地上的减速带。也不能太高，不然钩不住 Beam。

2. 形状： 要做成“V”形，这样 Beam 会自动滑到钩子中间，不会在半路上掉下来。

[图片：草图，画了一个V形的钩子，标注了离地高度]
记录人：印涵邸

会翻转的手腕 (Wrist)

为了拿颜色匹配分，我们需要把 Pin 翻过来。比如红 Pin 倒了，我们得把它正过来才能插进红 Goal。

所以，我们在爪子后面加了一个电机，让爪子可以旋转 180 度。

功能：
1. 抓取时：爪子朝下。
2. 翻转时：爪子朝上。
3. 释放时：爪子朝前。

怎么开车？(Controls)

宋书月设计了最顺手的按键方案：

左摇杆 + 右摇杆： 像开坦克一样，左手推左轮，右手推右轮。这样转弯最灵活！

L键 (L1/L2)： 控制手臂上下。左手食指控制，很方便。

R键 (R1/R2)： 控制爪子开合。右手食指控制。

F键： 控制手腕翻转。
